INSTRUCCIONES:

1. Abre Google Apps Script y publica tu Web App como:
   - Execute as: Me
   - Who has access: Anyone
2. Copia la URL del Web App.
3. Abre index.html y reemplaza la línea:
   const API = "PEGA_AQUI_URL_GOOGLE_SCRIPT";
   por tu URL del Web App.
4. Sube la carpeta boda-app a Netlify (arrastrando la carpeta).
5. Accede a la URL que Netlify te da.
6. Probar subiendo fotos/videos de prueba <2MB.
7. ¡Listo! Los invitados podrán subir fotos sin registrarse.
